Color System

Componente que modifica a cor do sistema, utilizado uma vez no sistema onde seu funcionamento altera todos os componentes no sistema instalados .

Componente em funcionamento:

Sobre a diretiva

Essa diretiva contém cores padrões para que seja feita a alteração de cor conforme o tema desejado.

Portanto, a utilização dessa diretiva fica condicionada apenas em utilizar variável de tema gerado pelo Localstorage.

Como Implementar?

instanciando o módulo:

Para inserir a diretiva de cores no HTML:

Deve se chamar a variável de localStorage no appComponent para instanciar no local o tema.

Não se esqueça de chamar no HTML geral (appComponent) a classe na estrutura do projeto para que a mudança ocorra.
