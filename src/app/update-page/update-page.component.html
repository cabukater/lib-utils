Atualizando


Atualize a versão do Angular CLI para versão atual da lib ( quando esse doc foi criado em 2021 a versão é 11.1.1, consulte a equipe de FRont End antes de fazer a atualização para uma versão diferente) Desinstale o Angular Cli

"instale novamente o Angular Cli (dessa forma fica mais efetiva a alteração)" ],
Agora atualize o Angular CDK
E também o Angular Core
Como utlizamos nossa lib alguns pacotes de dependências também deverão ser atualizadas no ptojeto para que possa ser utilizado a versão 11 do Angular como: Font Awesome

Atualização da Libshared

Considerações Para atualizarmos a lib (shared) é ideal que a versão seja especificada no arquivo package.json

shared-angular-components": "4.0.0" (ou superior)

(consulte no momento do update a versão mais atual da lib publicada e compatível com seu sistema)

Após a modificação do arquivo acima mencionado , na pasta root do projeto insira o seguinte comando no terminal:

rm -rf node_modules/ <= (Esse comando apaga a pasta node_modules/) Em seguida :' npm cache clean <= (Esse limpa o cache do npm)'">

Após os passos acima concluídos remova o arquivo package.lock.json Em seguida rode o comando no terminal 

  
   npm install <= esse comando instala novamente todos os pacotes para rodar no sistema

E finalmente rode o sistema

npm start ou ng serve npm 6.14.8.

Ao rodar o sistema podem ocorrer erro de injeções de dependencias ou de incompatibilidade de plugins e isso varia de projeto para projeto ( verifique qual o erro apontado pelo terminal e atualize ou ajuste a dependencia).
